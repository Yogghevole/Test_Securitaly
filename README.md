# Test Tecnico Securitaly - Gestione Noleggio DVD

Applicazione web per la gestione del noleggio DVD, sviluppata con **Laravel 12** (backend) e **React + Vite** (frontend).

## Tecnologie
- Backend: Laravel 12, PHP 8.5, MySQL 8.0
- Frontend: React 18, Vite, Ant Design, React Router, Axios
- Altro: Git, Composer, npm
